# Grepolis-Manager

The Grepolis Forum Manager (GFM) is a user script designed to enhance the forum management experience in the popular online strategy game, Grepolis. This script provides a convenient popup interface with a toolbar, buttons, and additional features such as a chatbox and speech functionality. It is particularly useful for alliance leaders and members who need to manage forums and topics efficiently.

Features
Popup Interface: A user-friendly popup with a toolbar and buttons for easy navigation.

Forum Management: Automates the creation of forums and topics based on predefined templates.

Player Information: Fetches and displays player information from the game.

Custom Styles: Injects custom styles to improve the visual appearance of the forum interface.

Toolbar Buttons: Includes buttons for various actions, such as creating forums, topics, and more.

Dynamic Content: Dynamically generates content based on the player's current context in the game.

How It Works
The script runs on Grepolis forums and provides a button to open a popup interface. From there, users can:

Create Forums and Topics: Automatically generate forums and topics using predefined templates.

Manage Player Information: Fetch and display player information stored in localStorage.

Navigate Forums: Easily navigate between different forums and topics within the game.

Customize Appearance: Apply custom styles to enhance the forum's visual appeal.

Installation
Install a userscript manager like Tampermonkey or Greasemonkey.

Create a new script and paste the contents of the Grepolis Forum Manager (GFM)-0.2 (2).user.js file.

Save the script and navigate to the Grepolis forum.

The script will automatically run, and you will see a new button to open the GFM popup.

Usage
Click the GFM button to open the popup interface.

Use the toolbar buttons to navigate and manage forums and topics.

Follow the on-screen instructions to create new forums and topics or fetch player information.
